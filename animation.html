CSS Animation

The Windmill Scene was a project learning about how CSS Animations work and how to be creative with them.

The challenge was learning how to manipulate the svg object that contained several paths representing all the elements that are part of this scene. Once I learned how to target the correct path I was able to apply animations to bring the scene alive.
